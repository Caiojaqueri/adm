# adm
Meu amigo pediu para eu criar um sistema que administrasse sua rotina de trabalho

1 - Gerenciador financeiro e de estoque de suas lojas.
Loja de ervas:
- cadastro de entrada de  mercadoria, peso total ex: 250g
- cálculo automático de quanto essa quantidade vai render em unidades de saquinhos de 10 g
ex: 250g / 10 = 25 unidades
- saída de mercadoria (por unidade)
- lembrede de estoque acabando
- quanto entra de valor.

Loja de produtos de dez:
- cadastro de entrada de mercadoria (uni)
- saída de mercadoria (por unidade)
- lembrete de estoque acabando
- quanto entra de valor
Mostrando o histórico de entradas e saídas sempre!

2 - Organização de parcerias
- cadastro de nome, função e dias de postagem de conteúdo
- organizado em tabela
- pastas de cada parceria para armazenar documentos

3 - agenda de compromissos das parcerias
- agenda para marcar os dias e horário de cada parceria
- lembrete de agendamento
ex: seg parceria 2, ter parceria 2, até sábado

Tudo bastante simples e prático de utilizar! 
Vou utilizar Html, Css e Js para o Frontend. Python, Banco SQLite e FastAPI para o Backend.
